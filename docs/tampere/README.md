# Publika > tampere

Tampere public transport schedule times module for MirrorMirror project

![Module](fkvgxtfd.png)

Quick config example:

```js
{
  module: "publika",
  position: "top_left",
  config: {
    feed: "tampere",
    stops: [
      "0807",
      "0518"
    ]
  }
}
```
