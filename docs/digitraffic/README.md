# Publika > digitraffic

VR train schedule times module for MirrorMirror project

![Module](fktthyam.png)

Quick config example:

```js
{
  module: "publika",
  position: "top_left",
  config: {
    feed: "digitraffic",
    stops: [
      { id: "PSL", type: "station" }
    ]
  }
}
```
